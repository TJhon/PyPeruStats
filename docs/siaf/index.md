# SIAF — Public Expenditure

!!! warning "Documentation in progress"
    This page is a placeholder. Full documentation for `perustats.siaf` will be added when the source code is finalized.

The `perustats.siaf` module will provide programmatic access to Peru's **Sistema Integrado de Administración Financiera (SIAF-SP)** data, published by the Ministerio de Economía y Finanzas (MEF).

---

## Planned Features

- Download public budget execution data by government level (national, regional, local).
- Filter by year, function, program, and executing unit.
- Automatic normalization of MEF's inconsistent field formats.
- Output as tidy `pandas` or `polars` DataFrames.
